# TachyonVisualization

Visualization of tachyon condensation on an open string.

## Install

```bash
pip install -e .
```

## Run

```bash
tachyon-firewall
```

## Options

```
--string-length   String length L
--gamma           Tachyon instability rate
--damping         Damping coefficient
--eps0            Initial perturbation amplitude
--sign {1.0,-1.0} Which vacuum to roll into
--t-total         Total physical time
--fps             Animation frame rate
--nx              Number of spatial grid points
```
